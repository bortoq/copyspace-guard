# Pricing draft

## Diagnostic Audit — $9,500

Best for first customer conversation.

Includes:

- one workload;
- one CSV/JSON input;
- STRICT1 model;
- baseline vs candidate report;
- one review call;
- CI-gate recommendation.

## Optimization Sprint — $35,000

Best when the diagnostic finds a meaningful gap.

Includes:

- up to three workloads;
- custom importer;
- compare customer schedule vs candidate schedules;
- threshold design for CI;
- model extension plan.

## Enterprise Pilot — $75,000+

Best for AI infra, HPC, storage vendor or database vendor.

Includes:

- on-prem execution package;
- private adapters;
- integration with CI/dashboard;
- security review support;
- cost-attribution roadmap.

## Annual license — $30,000–200,000/year

Depends on:

- number of teams;
- number of workloads;
- support SLA;
- private adapters;
- on-prem requirements;
- vendor/OEM rights.
